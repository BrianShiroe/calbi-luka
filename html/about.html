Luka CCTV Monitoring System

Advanced real-time object detection and alert system for safety monitoring

Key Features

Multi-Source Monitoring

Supports RTSP, HTTP streams, and YouTube live feeds with automatic reconnection capability for uninterrupted surveillance.

AI-Powered Detection

YOLO-based object detection for hazards like fire, smoke, crashes, floods, and landslides with configurable confidence levels.

Smart Alert System

Configurable audio alerts with volume control, duration settings, and visual notifications for detected hazards.

Evidence Recording

Automatically saves images of detected events with timestamps and metadata for later review.

Performance Optimization

Frame skipping, resolution adjustment, and GPU acceleration for efficient resource usage.

Technical Specifications

  • Backend:   Python Flask server with SQLite database
  • AI Model:   Ultralytics YOLO
  • Configurable Settings:   Resolution (144p-1080p), frame rate, detection sensitivity
  • Performance Metrics:   Real-time FPS, processing time, CPU usage monitoring
  • Stream Handling:   Supports multiple concurrent streams with individual controls
  • Alert Management:   Event logging with resolution tracking and historical data

System Version: 1.0.0 | Powered by Python, Flask, YOLO, and OpenCV
